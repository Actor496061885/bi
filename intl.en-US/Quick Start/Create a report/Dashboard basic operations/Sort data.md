# Sort data {#task_a2c_1jy_5db .task}

On the Data tab, you can sort data based on the specified dimension and measurement fields. The chart shows the data based on the selected sort result.

1.   Select a field, such as **order\_number**, and click the up or down arrow behind the field. See the following figure. 

    The up arrow indicates the ascending order, and the down arrow indicates the descending order.

    ![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/9116/15332612541470_en-US.png)

2.   Click **Update**.
