# Add an organization member. {#task_yfq_jx1_5db .task}

You can add members by using a primary Alibaba Cloud account or a RAM sub-account. If you need add more members, you can add members by using the function of adding members in batch.

## Adding member through Alibaba Cloud account {#task_cjs_gy1_5db}

1.   Log on to the Quick BI console. 
2.   Select **Settings** \> **Org Units** \> **Organization Members**. 
3.   Click **Add Member**, as shown in the following figure.![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/9154/1065_en-US.png)

 
4.   Click **Add Alibaba Cloud ID** tab. 
5.   Enter an Alibaba Cloud ID and a nick name manually, and check whether the member is set as an organizational administrator.![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/9154/1066_en-US.png)

 
6.   Click **OK** to complete the member adding. 

## Adding members through a RAM sub-account {#task_a4d_fkc_5db}

1.   Log on to the Quick BI console. 
2.   Select **Settings** \> **Org Units** \> **Organization Members** \> **Add Member**. 
3.   Click the **RAM User** tab. 
4.   Enter a RAM sub-account ID and a nick name manually, and check whether the member is set as an organizational administrator. 

    **Note:** The Alibaba Cloud account should be the corresponding cloud account \(non-account ID\) for the primary account \).

5.   Click **OK** to complete the member adding. 

## Adding members in batch {#task_zqz_xjc_5db}

1.   Log on to the Quick BI console. 
2.   Select **Settings** \> **Org Units** \> **Organization Members**. 
3.   Click **Import Members**, as shown in the following figure.![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/9154/1084_en-US.png)

 
4.   Click **Select a file** and select a local file.![](http://static-aliyun-doc.oss-cn-hangzhou.aliyuncs.com/assets/img/9154/1085_en-US.png)

 
5.   Click **OK** to complete member adding in batch.
